# GitHub Integration Tools

This tool category enables comprehensive interaction with the GitHub platform. It supports repository management, issue tracking, branch control, commit history inspection, and pull request lifecycle operations.

## Available Tools

### Repositories
- **`search_repositories`**: Search for GitHub repositories.
- **`create_repository`**: Create a new repository.
- **`fork_repository`**: Fork a repository to your account.

### File Operations
- **`get_file_contents`**: Retrieve contents of files in a repository.
- **`create_or_update_file`**: Create or modify a single file.
- **`push_files`**: Push multiple files to a repository in a single commit.

### Issues
- **`create_issue`**: Open a new issue.
- **`list_issues`**: Retrieve issues in a repository.
- **`get_issue`**: Get details of a single issue.
- **`update_issue`**: Edit issue state, labels, or content.
- **`add_issue_comment`**: Post comment on an issue.
- **`search_issues`**: Search issues across repositories.

### Pull Requests
- **`create_pull_request`**: Open a new pull request.
- **`get_pull_request`**: Get details of a pull request.
- **`list_pull_requests`**: List pull requests in a repository.
- **`merge_pull_request`**: Merge a pull request.
- **`create_pull_request_review`**: Submit review on a pull request.
- **`get_pull_request_files`**: Retrieve files modified in a pull request.
- **`get_pull_request_status`**: Get build/CI status of a pull request.
- **`get_pull_request_comments`**: Retrieve code comments on a pull request.
- **`get_pull_request_reviews`**: Retrieve reviews on a pull request.
- **`update_pull_request_branch`**: Update pull request branch with base branch.

### Branches & Commits
- **`create_branch`**: Create a new branch.
- **`list_commits`**: List commits in a repository or branch.

### Search
- **`search_code`**: Search code across GitHub.
- **`search_users`**: Search GitHub users.
