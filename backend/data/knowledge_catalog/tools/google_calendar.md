# Google Calendar Event & Schedule Management

These tools provide complete programmatic access to Google Calendar functionality, enabling comprehensive schedule and event management.

**Available Tools:**

- **`calendar`**: Manage Calendar.app events - create events or list today's scheduled events
- **`list-calendars`**: Discover all available calendars associated with the account
- **`list-events`**: Retrieve events from one or more calendars using IDs or names
- **`search-events`**: Find events by text query across calendar data
- **`get-event`**: Retrieve detailed information about a specific event by ID
- **`create-event`**: Create a single new calendar event
- **`create-events`**: Bulk create multiple events with shared defaults and per-event overrides
- **`update-event`**: Modify existing events including recurring event series
- **`delete-event`**: Remove calendar events from the schedule
- **`respond-to-event`**: Accept, decline, or mark tentative responses to event invitations
