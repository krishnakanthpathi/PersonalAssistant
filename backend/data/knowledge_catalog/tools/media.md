# YouTube Media Search, Transcript & Download

This tool group provides comprehensive access to YouTube media content. It enables searching for videos by query, retrieving transcripts for accessibility or analysis, and downloading videos for offline use.

**Available Tools:**
- **`search_youtube_videos`**: Find YouTube videos using search terms and keywords.
- **`get_youtube_transcript`**: Extract the full text transcript from a YouTube video.
- **`download_youtube_video`**: Download a YouTube video file to local storage.
