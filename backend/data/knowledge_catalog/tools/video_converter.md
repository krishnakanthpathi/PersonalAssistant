# Video Converter Tools

This tool category handles local video format conversions and codec operations via FFmpeg utility commands on the host machine.

## Available Tools

### Verification & Support
- **`check_ffmpeg_installed`**: Check if the host system has FFmpeg installed.
- **`get_supported_formats`**: List supported output formats and codec containers.

### Conversion Operations
- **`convert_video`**: Convert a video file to another format or change container types.
