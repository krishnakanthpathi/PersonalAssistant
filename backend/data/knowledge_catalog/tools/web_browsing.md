# Web Browsing, Scraping & Research Tools

This category provides a complete toolkit for interacting with the web, from simple page scraping to complex autonomous research. Tools include:

- **firecrawl_scrape**: Extract content from a single URL with support for markdown, JSON, branding formats, and JavaScript-rendered pages.
- **firecrawl_map**: Discover all indexed URLs on a website; ideal for finding the right page before scraping.
- **firecrawl_search**: Powerful web search with optional content extraction from results.
- **firecrawl_search_feedback**: Submit feedback on search results to improve quality and earn credits.
- **firecrawl_feedback**: Generic feedback endpoint for scrape, parse, map, or search jobs.
- **firecrawl_crawl**: Multi-page crawling with depth and limit controls.
- **firecrawl_check_crawl_status**: Check the status of an ongoing crawl job.
- **firecrawl_extract**: LLM-powered structured data extraction from multiple URLs.
- **firecrawl_agent**: Autonomous AI agent that browses, searches, and extracts data independently.
- **firecrawl_agent_status**: Poll for agent job results.
- **firecrawl_interact**: Interact with pages in a live browser session (click, fill forms, navigate).
- **firecrawl_interact_stop**: Stop an active interact session.
- **firecrawl_parse**: Parse local documents (PDF, Word, Excel, HTML) into markdown or structured JSON.
- **firecrawl_monitor_create**: Create recurring monitors for page changes or search results.
- **firecrawl_monitor_list/get/update/delete/run**: Monitor management operations.
- **firecrawl_monitor_checks/check**: View monitor check history and diff results.
- **firecrawl_research_search_papers**: Find research papers by topic across academic literature.
- **firecrawl_research_inspect_paper**: Fetch full metadata for a specific paper.
- **firecrawl_research_related_papers**: Discover related papers via citation graph.
- **firecrawl_research_read_paper**: Read full-text passages from a paper.
- **firecrawl_research_search_github**: Search GitHub issues, PRs, and readmes.
