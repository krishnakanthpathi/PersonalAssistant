# Network Utilities & Database Connectivity

The **Network Utilities** category provides essential interfaces for interacting with external data services over a network. These tools facilitate direct connections to database instances, allowing for real-time data retrieval and manipulation.

### Available Tools
- **`query_rds_database`**: Establishes a connection to a PostgreSQL or MySQL database via a connection string URL and executes SQL queries (e.g., SELECT statements) to retrieve or filter data.
