# macOS Finder & System Shortcuts Integration

These tools enable programmatic control over macOS Finder operations and Apple Shortcuts, facilitating file system maintenance and automated workflows.

**Available Tools:**
- **`empty_trash`**: Empties the macOS Trash folder
- **`shortcut_list`**: List Apple Shortcuts on this Mac (optional folder filter)
- **`shortcut_run`**: Run an Apple Shortcut by name (optional input/output paths). 60 s timeout
