# macOS Finder & File Management Tools

These tools provide comprehensive integration with the macOS Finder and system-level file operations:

- **`empty_trash`**: Empties the macOS Trash folder permanently.
- **`reveal_in_finder`**: Opens Finder and highlights a specified file or directory.
- **`get_finder_selection`**: Retrieves the currently selected items in Finder as an array of file URLs.
- **`set_finder_tags`**: Applies or modifies Finder tags on one or more files for organization.
- **`quick_look`**: Launches a Quick Look preview for a file path without blocking the workflow.
- **`move_to_trash`**: Moves a file or directory to the Trash (recoverable until emptied).
- **`spotlight_search`**: Performs a Spotlight search using NSMetadataQuery/mdfind and returns matching file paths.
- **`shortcut_list`**: Lists available Apple Shortcuts on the Mac, with optional folder filtering.
- **`shortcut_run`**: Executes an Apple Shortcut by name with optional input/output paths (60-second timeout).
- **`trash_thread`**: Moves a thread to the trash.
- **`untrash_thread`**: Removes a thread from the trash.
