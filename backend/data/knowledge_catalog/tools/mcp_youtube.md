# MCP Server - youtube

Integrated MCP server providing tools and capabilities for youtube.

### Available Tools

- **`search_youtube_videos`**: Search YouTube videos based on a search term.
- **`get_youtube_transcript`**: Get the transcript of a YouTube video.
- **`download_youtube_video`**: Download a YouTube video.
- **`play_youtube_audio`**: Play the audio of a YouTube video or search query in the background using mpv.
- **`stop_youtube_audio`**: Stop the currently playing background audio.

Returns:
    A message confirming playback was stopped.
